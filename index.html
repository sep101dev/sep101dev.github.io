Starting things off with Mark's Scratch game (Unit 1), this is the first time we were exposed to coding logic. Scratch is a block-based programming language and website, and is very common for novice programmers to use. With it comes the simplest coding logic, like booleans, and conditonals, but also some more advanced topics such as functions and arithmetics.

Within Mark's project is a muli-level maze game, in which first you must go through a door to find a golden apple, when you touch the golden apple, the size of your sprite decreases, allowing you to go through the door you intially couldn't and go to the next level. In the second cave is a plethroa of AI, some who have a set motion, while others go to random spots at random speeds. The concept was inspired off of Pac-Man. In the final cave is a giant crab, who is programmed to constantly follow you, and as the timer goes down, he gets faster. Eventually around the end of the timer, when it seems as though the crab is going too fast, an orange colored apple appears and when touched makes the main sprite invincible, giving you the choice to defeat the final boss of the game. There are many details that I did not include within this paragraph, but that is only so you play the game yourself, inside, and see what other sprites and AI are in this game. This is Mark's Python + Turtle project (Unit 3)

Turtle is a library in the popular coidng laguage Python, it's used for geometric shapes, and programs that require drawings. Within the context of our class, we used it on the website repl.it, and learned to create drawings. In my project, I've created a scenery of the summer sky, the bright sun and it's orange rays, with 3 cyan clouds above it. The hardest part of this project were definitely trying to understand the concept that is geometric shapes and how they are formed, especially as someone who hasn't taken geometry class yet. This is also the first instance of using an actual programming language. We learned the concept of the while loop, similar to the forever loop in Scratch.
