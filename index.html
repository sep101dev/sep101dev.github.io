Our Digital Portfolio (SEP 101)

Made by Mark and Anthony

Starting things off with Mark's Scratch game (Unit 1), this is the first time we were exposed to coding logic. Scratch is a block-based programming language and website and is very common for novice programmers. With it comes the simplest coding logic, like booleans, and conditionals, but also some more advanced topics such as functions and arithmetics.

Within Mark's project is a multi-level maze game, in which first you must go through a door to find a golden apple, when you touch the golden apple, the size of your sprite decreases, allowing you to go through the door you initially couldn't and go to the next level. In the second cave is a plethora of AI, some of who have a set motion, while others go to random spots at random speeds. The concept was inspired by Pac-Man. In the final cave is a giant crab, who is programmed to constantly follow you, and as the timer goes down, he gets faster. Eventually, around the end of the timer, when it seems as though the crab is going too fast, an orange-colored apple appears and when touched makes the main sprite invincible, giving you the choice to defeat the final boss of the game. There are many details that I did not include within this paragraph, but that is only so you play the game yourself, inside, and see what other sprites and AI are in this game. Anthony’s scratch game is taking a simple class assignment to the next level. The game is a 2D shooter, in which there is 1 tutorial level and three main levels. Turtle is a library in the popular coding language Python, it's used for geometric shapes, and programs that require drawings. In my project, I've created a scenery of the summer sky, the bright sun, and its orange rays, with 3 cyan clouds above it. The project is a seasonal calendar. It allows you to check what season it is outside and what month it is via LEDs.… or that’s how it was supposed to go. The hardware side of the project did indeed work, however the software side, which I(Mark) was tasked with, didn’t go so well.
